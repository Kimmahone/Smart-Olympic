## 📖 프로젝트 소개
**스마트 올림픽**은 초등학교 체육 수업(건강 및 도전 영역)을 위해 개발된 웹 기반 AI 모션 인식 프로그램입니다. 학생들의 신체 활동량을 늘리고, 자기주도적인 체력 단련을 게임처럼 즐겁게 할 수 있도록 기획되었습니다. 

별도의 앱 설치나 회원가입 없이, 교실에 있는 **아이패드, 크롬북 등 스마트기기의 웹 브라우저와 카메라**만 있으면 누구나 즉시 6가지 다채로운 체육 종목에 참여할 수 있습니다.

---

## ✨ 주요 기능 및 특징

### 1. 🛡️ 100% 온디바이스(On-device) 개인정보 보호
* **No Server, No DB:** 학생의 카메라 영상 및 이름 데이터는 외부 서버로 단 1바이트도 전송되지 않습니다.
* **Privacy by Design:** 오직 웹 브라우저 메모리 내에서만 실시간으로 연산되며, 브라우저 혹은 새로고침 시 데이터가 즉시 파기되어 학교 현장에서 개인정보 유출 걱정 없이 안전하게 사용할 수 있습니다.

### 2. 🏃‍♂️ 6대 체력 요소를 반영한 '꼼수 방지' 맞춤형 종목
아이들의 흥미를 끌어올리면서도 정확한 자세를 유도하기 위해 정교한 인식 알고리즘을 적용했습니다.
* **100m 달리기 (순발력):** 무릎을 확실히 들어 올리는 발구름 폭을 계산하여 전진 (하단 진행률 바 제공)
* **50m 수영 (심폐지구력):** 팔꿈치를 굽히는 꼼수 방지! 팔을 쫙 펴고 크게 원을 그리는 '풍차돌리기' 자세를 취할 경우 +3m 보너스 이동
* **30초 체조 (근지구력):** 팔벌려뛰기 4박자(옆-차렷-위-차렷)를 정확히 수행할 때만 카운트 및 성공 효과음(`ding!`) 출력
* **30초 복싱 (민첩성):** 무작위로 나타나는 타겟을 향해 뻗어 나가는 주먹의 타격 인식
* **송판 격파 (유연성):** 발목의 이동 '속도(Velocity)'를 측정하여, 일정 파워 이상으로 가격해야만 송판이 깨지는 파티클 효과 제공
* **서핑 밸런스 (평형성):** 어깨의 무게 중심 이동을 통해 상어를 피하고 동전을 획득하는 아케이드 모드 (시간 비례 속도 증가)

### 3. 🎥 마이크 목소리까지 담기는 영상 기록증 발급
* Canvas API의 실시간 렌더링과 Web Audio API를 결합하여 **오디오 믹싱 기술**을 구현했습니다.
* 게임의 신나는 BGM, 타격 효과음은 물론, **선생님과 친구들의 열띤 응원 목소리(마이크 증폭)**까지 하나의 비디오 파일(`webm`/`mp4`)로 병합하여 저장합니다.
* 경기 종료 후 학생 이름이 적힌 '나만의 기록증 이미지'를 로컬 기기로 다운로드할 수 있습니다.

### 4. 📱 교실 스마트기기에 최적화된 UI/UX
* **태블릿 친화적:** 게임이 시작되면 상단 타이틀이 자동으로 숨겨져 작은 태블릿 화면에서도 점수와 영상이 가려지지 않습니다.
* **500ms 카운트다운:** 아이들의 텐션을 올리기 위해 박진감 넘치는 0.5초 단위 카운트다운과 BGM 딜레이 재생을 적용했습니다.
* **원터치 초기화:** 화면 엉킴 없이 항상 쾌적한 플레이가 가능하도록 '중단하기' 및 '다시하기' 클릭 시 메모리를 완벽하게 초기화(`reload`)합니다.

## 🛠️ 기술 스택 (Tech Stack)
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **AI Model:** [Google MediaPipe Pose](https://google.github.io/mediapipe/solutions/pose.html) (인체 뼈대 랜드마크 실시간 추출)
* **Media & Graphics:** HTML5 `<canvas>`, Web Audio API, `MediaRecorder` API
* **Deployment:** Vercel

## 💻 로컬 실행 방법 (Getting Started)
이 프로젝트를 로컬 환경에서 실행하고 테스트하려면 다음 단계를 따르세요.

1. 저장소를 클론(Clone)합니다.
```bash
git clone [https://github.com/Kimmahone/smart-olympic.git](https://github.com/Kimmahone/smart-olympic.git)
```
2.프로젝트 폴더로 이동합니다.
```bash
cd smart-olympic
```
3. 웹 서버 환경에서 실행합니다. (카메라 권한을 얻기 위해 로컬 서버가 필요합니다.)
 - VS Code 사용자: Live Server 익스텐션을 사용하여 index.html을 엽니다.
 - Python 사용자: python -m http.server 8000 실행 후 브라우저에서 localhost:8000 접속
